# German Peer Review Dataset

Data files can be downloaded from this [form](data/) and are released under a non-commercial license. Place the files directly in this folder (`\data`).  

The five provided files are:
- `reviews_2015.csv`
- `reviews_2016.csv`
- `reviews_2017.csv`
- `reviews_2018.csv`
- `reviews_2019.csv`
